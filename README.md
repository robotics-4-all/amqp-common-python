# amqp-common-python
Higher-level features for AMQP, such as PubSub and RPC (server-client).

# Installation

**CURRENTLY ONLY PYTHON2 IS SUPPORTED**

```bash
python2 setup.py install --user
```

or

```bash
pip install . --user
```

# Examples

Look at the `examples` folder. Includes PubSub and RPC examples.
